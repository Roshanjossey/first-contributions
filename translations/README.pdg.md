# First Contributions

Dis small project na to epp anybodi wey just dey start or persin wey dey struggle to follow make dia own first contribution to dis open source waka. if na to follow wee people make ya ogbonge first contribution, yhu go nid to follow d way I wan show yhu. Just check under dis yans.

    If yhu no fit use command line wit ya chest, OR e be like say command line dey cuz head to pain yhu, [Abeg yhu for like reason dis GUI tools  tutorials. E nor hard at all, E be like ABC](#tutorials-using-other-tools)

<img align="right" width="300" src="https://firstcontributions.github.io/assets/Readme/fork.png" alt="fork this repository" />

#### If like dis git nor dey ya lappy/computer, [Yhu go nid install am](https://help.github.com/articles/set-up-git/).

## Fork de repository

If yhu wan fork dis repo, yhu go nid to press dat fork button wey dey for de top-right of dis page. [Nor laff 'fork' no be bad tin.]
As yhu fork finish, na so yhu go com get ya own copy of dis repo for ya github akant.

## Clone de repository

<img align="right" width="300" src="https://firstcontributions.github.io/assets/Readme/clone.png" alt="clone this repository" />

As yhu fork finish, na to com clone de forked repo to ya lappy/desktop. Go ya Github akant, open dat repo wey yhu fork, press de code button wey dey de top-right, den press de _copy to clipboard_ icon.

Sharperly... Open one terminal so yhu go fit run some git commands:

```
git clone "dat link wey yhu just copy"
```

Rememba say "dat link wey yhu just copy" (abeg nor put de quotation marks) na de link to dis repo (ya forked copy of dis project). If e be like say de mata wan confuse yhu, just try check wetin I yan before wee reach hia, so yhu go fit get de link.

<img align="right" width="300" src="https://firstcontributions.github.io/assets/Readme/copy-to-clipboard.png" alt="copy URL to clipboard" />

Make I give yhu ogbonge example:

```
git clone https://github.com/na-me-be-dis/first-contributions.git
```

Wia `na-me-be-dis` na ya GitHub username. For hia, yhu dey copy all d tinz wey dey inside de first-contributions repository for GitHub to ya own lappy(Computer).

## Na to create branch

If yhu nor dey inside de repo directory for ya lappy/computer, yhu for like change am oo... make wahala (error) no for burst (show).

```
cd first-contributions
```

Yhu go nid create wetin dem dey call (branch). Yhu go use dis `git checkout` command to create am:

```
git checkout -b ya-new-branch-name
```

Make I give yhu anoda ogbonge example:

```
git checkout -b add-pidgin-sweet
```

(No be must say yhu must join the word _add_ for de name of de new branch, but e dey nice to write am join bcoz na coded tin to do. As e be say yhu wan add ya name to de list.)

## Make we make some kind changes and commit dem

Yhu go need open de `Contributors.md` file for wetin dem dey call (text editor), add ya name for inside. Abeg nor add am for de starting or last part of de file. Just put am anywia for de middle. Yhu go com save de file, so dat ya name go follow gum body with de oda names wey dey inside de file.

<img align="right" width="450" src="https://firstcontributions.github.io/assets/Readme/git-status.png" alt="git status" />

To take ya eye see wetin yhu don do, just go de project directory for ya terminal, den type dis git command `git status`, yhu go see say some tinz don change.

Yhu go nid add dos changes wey yhu don make for dat new branch wey yhu bin create. To do am, yhu go use de `git add` command:

```
git add Contributors.md
```

Oya na, as yhu don add de changes finish, na to com commit de changes. To commit am, yhu go use de `git commit` command:

```
git commit -m "Add <ya-name> to Contributors list"
```

Sharp sharp change <ya-name> with de ogbonge name wey ya Papa & Mama give yhu. OR as e be say de world don spoil, yhu fit put ya nickname or anytin wey fit define yhu.

## Oya Push de tinz wey don change to GitHub

As we don dey round up like dis, yhu go nid Push ya changes to de new banch wey yhu bin create. To do am, yhu go use de command `git push`:

```
git push origin <add-dat-ya-new-branch-name>
```

Sharp sharp change <add-dat-ya-new-branch-name> with de name of dat branch wey yhu create before before.

## Submit ya changes make for dem for torchlight am

As yhu enta ya repository on GitHub, yhu go see one `Compare & pull request` button. Press dat button.

<img style="float: right;" src="https://firstcontributions.github.io/assets/Readme/compare-and-pull.png" alt="create a pull request" />

Oya oo! Time don reach make yhu for submit de pull request (PR).

<img style="float: right;" src="https://firstcontributions.github.io/assets/Readme/submit-pull-request.png" alt="submit pull request" />

As time no dey wait for perzin, Na so I go dey merge ya changes to gum body with de master branch of dis project. Sharp! sharp!! As I succeed to merge de changes, yhu go recieve email alert.

## Wia man pikin go chuk head from hia?

Congratulobia! yhu don succeed for dis Open Source-Github-life. Just now now, yhu don finish de ogbonge _fork -> clone -> edit -> pull request_ workflow wey yhu go dey jam steady as contibutor wey yhu be.

Omo! Make pple nor for hear word oo. E nor easy like dat... Yhu fit share de celebration wit ya friends and disciples (followers) by presin dis link. [web app](https://firstcontributions.github.io/#social-share).

As e dey sweet yhu for bodi, yhu fit join our slack team for say yhu nid any epp OR e be say yhu wan ask ogbonge queshions. [Join slack team](https://join.slack.com/t/firstcontributors/shared_invite/zt-iywfifau-_aMtdwTjBoMzQqzW8~YUUA).

Oya! as e dey hot and e be say yhu wan level up, make we show yhu road to contribute to oda projects. We don gada some corrct projects wey get small small yawa wey yhu fit use hold ya side. No dullin, check am out [the list of projects in the web app](https://firstcontributions.github.io/#project-list).
