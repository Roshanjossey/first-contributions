# Loyihaga birinchi hissa qo'shing (First Contributions)

Bu qiyin. Birinchidan, nimadir qilish har doim ham qiyin. Xatolar qilish juda yoqimsiz, ayniqsa jamoada ishlasangiz. Barcha ochiq manbali loyihalar hamkorlik va noyob ishlardan iborat. Biz boshlang'ich dasturchilar uchun o'rganish va hamkorlikdagi dastlabki qadamlarni osonlashtirmoqchimiz.

Siz har doim darsliklar va maqolalarni o'qishingiz mumkin, ammo amaliy sharoitda ishlashdan yaxshiroq nima bor? Ushbu loyihaning maqsadi yoshlarni to'g'ri yo'lga yo'naltirish, shuningdek ularga birinchi hissa qo'shish imkoniyatini berishdir. Agar siz birinchi hissangizni qo'shish imkoniyatini qidirsangiz, quyidagi oddiy amallarni bajaring. Biz qiziqarli bo'lishiga va'da beramiz.

<img align="right" width="300" src="https://firstcontributions.github.io/assets/Readme/fork.png" alt="fork this repository" />


Agar kompyuteringizda git yo'q bo'lsa, [ uni qo'shing ]( https://help.github.com/articles/set-up-git/ )

## Ushbu repositoryani fork qiling

Ushbu repositoryani fork qilish uchun sahifani yuqorisidagi `Fork` tugmasini bosing. Shunda siz ushbu repositorya nusxasini o'z akauntingizda yaratasiz.

## Repositoriyani klonlash

<img align="right" width="300" src="https://firstcontributions.github.io/assets/Readme/clone.png" alt="clone this repository" />

Endi ushbu fork qilingan repositoryani shaxsiy kompyuteringizga joylashtiring. Havolani nusxalash uchun "clone" tugmachasini bosing, so'ngra "copy to clipboard" belgisini bosing.

Terminalni oching va quyidagi git buyrug'ini bajaring:

```
git clone "url you just copied"
```
Bu erda "url you just copied" (qo'sh tirnoqsiz) sizning repositoryangizga havola. Ushbu havolani olish uchun avvalgi qadamlarni ko'ring.

<img align="right" width="300" src="https://firstcontributions.github.io/assets/Readme/copy-to-clipboard.png" alt="copy URL to clipboard" />

Masalan:
```
git clone https://github.com/this-is-you/first-contributions.git
```
Bu yerda `this-is-you` sizning github-dagi username-ingiz. Shunday qilib GitHub-dan sizning kompyuterningizga 'first-contributions' repositoryani ko'chirasiz.

## Branch yaratish

Agar siz hali u erda bo'lmasangiz, kompyuteringizdagi repositorya katalogiga o'ting.

```
cd first-contributions
```
Endi `git checkout` komandani yordami bilan branch quring.

```
git checkout -b <add-your-name>
```

Masalan:
```
git checkout -b add-alonzo-church
```
(Branch nomi uchun unga *add* qo'shilishi shart emas, lekin uni kiritish maqsadga muvofiqdir, chunki bu branchning maqsadi sizning ismingizni ro'yxatga qo'shishdir.)

## Kerakli o'zgarishlarni amalga oshiring va ularni commit qiling

Matn muharririda `Contributors.md` faylini oching, ismingizni kiriting va faylni saqlang. Agar siz loyiha katalogiga kirsangiz va `git status` ni bajarsangiz, o'zgarishlarni ko'rasiz. Ushbu o'zgarishlarni `git add` buyrug'i bilan qo'shing.

```
git add Contributors.md
```
Endi ushbu o'zgarishlarni `git commit` buyrug'i bilan commit qilasiz.

```
git commit -m "Add <your-name> to Contributors list"
```
`<your-name>` ni o'z ismingiz bilan almashtiring

## O'zgarishlarni github-ga joylashtiring

O'zgarishlarni `git push` buyrug'i bilan joylashtiring
```
git push origin <add-your-name>
```
`<Add-your-name>` nomini avval yaratilgan branch nomiga o'zgartiring.

## Ko'rib chiqish uchun o'zgarishlarni tasdiqlang

Agar GitHub-dagi repositoryaga kirsangiz, `Compare & pull request` tugmachasini ko'rasiz. Uni bosing.

<img style="float: right;" src="https://firstcontributions.github.io/assets/Readme/compare-and-pull.png" alt="create a pull request" />

Endi pull-rekvizitini tasdiqlang.

<img style="float: right;" src="https://firstcontributions.github.io/assets/Readme/submit-pull-request.png" alt="submit pull request" />

Yaqinda men barcha o'zgarishlarni ushbu loyihaning asosiy qismi bilan birlashtiraman. O'zgarishlar qabul qilinganda sizga elektron pochta xabarlari keladi.
